Second year university project with 5 people.
It's a shell scripting project (100% Bash/Shell) focused on virtualization and infrastructure automation.
What it does:

1.ansiInstall.sh — installs Ansible
2.vault.sh — sets up Ansible Vault (for secrets/passwords)
3.vmpackage.sh — installs VM packages
4-6. scripts — creates and configures individual virtual machines (named natovm, viliausvm, dovydovm, arnovm — likely team members' VMs)
7.hostas.sh — configures the host machine
8.dockeris.sh — sets up Docker
9.playbookai.sh — runs Ansible playbooks
db_setup.sh — sets up a database
webstack.sh — sets up a web stack

In short: it's a group project that automates the creation and configuration of multiple virtual machines using Ansible + Docker + Shell scripts, likely for a school or college virtualization/DevOps class. Each team member has their own VM setup script.
